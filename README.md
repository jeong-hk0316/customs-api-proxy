# 한국 관세청 API 프록시 서버

## 설명
관세청 공공데이터를 GPTs에서 사용하기 위한 프록시 서버

## 주요 기능
- HS코드별 수입 통계 분석
- 국가별 수입량 및 금액 조회  
- 수입 시점 환율 적용 원화 환산
- 국가별 수입 비율 계산
- 관세율 정보 제공

## 사용 방법
/api/import-analysis?hsSgn=0202&strtYymm=202301&endYymm=202312

## 응답 예시
- 총 수입량 (kg)
- 총 수입금액 (달러)
- 총 수입금액 (원화)
- 국가별 수입 비율
- 국가별 상세 정보
- 관세율 정보
